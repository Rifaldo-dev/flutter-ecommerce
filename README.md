# E-Commerce Flutter Application

> Aplikasi E-Commerce mobile menggunakan Flutter dan Supabase  
> Memenuhi Ujian Akhir pada mata kuliah Pemrograman II (dua)

## <img src="https://img.icons8.com/fluency/48/000000/info.png" width="20"/> Tentang Aplikasi

Aplikasi e-commerce modern yang dibangun dengan Flutter untuk platform mobile (Android & iOS) dan web. Aplikasi ini menyediakan fitur lengkap untuk berbelanja online dengan antarmuka yang responsif dan user-friendly.

## <img src="https://img.icons8.com/fluency/48/000000/rocket.png" width="20"/> Tech Stack

- <img src="https://img.icons8.com/color/48/000000/flutter.png" width="16"/> **Framework**: Flutter 3.35.6
- <img src="https://img.icons8.com/color/48/000000/postgreesql.png" width="16"/> **Backend**: Supabase (PostgreSQL)
- <img src="https://img.icons8.com/color/48/000000/code.png" width="16"/> **State Management**: GetX
- <img src="https://img.icons8.com/color/48/000000/database.png" width="16"/> **Local Storage**: GetStorage
- <img src="https://img.icons8.com/color/48/000000/material-ui.png" width="16"/> **UI Components**: Material Design 3
- <img src="https://img.icons8.com/color/48/000000/animation.png" width="16"/> **Animations**: Lottie
- <img src="https://img.icons8.com/color/48/000000/google-fonts.png" width="16"/> **Fonts**: Google Fonts (Poppins)
- <img src="https://img.icons8.com/color/48/000000/settings.png" width="16"/> **Environment**: flutter_dotenv
